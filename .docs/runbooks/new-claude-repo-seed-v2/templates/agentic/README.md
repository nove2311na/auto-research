# Agentic System

This folder stores the durable parts of the agentic repo.

| Folder | Purpose |
|---|---|
| `knowledge/` | Stable project facts and source indexes. |
| `memory/` | Memory candidates and approved memory cards. |
| `policies/` | Tool risk, approval, and safety policies. |
| `orchestration/` | Workflow and handoff contracts. |
| `specs/` | Generated `agent_system_spec` and scaffold plans. |
| `evals/` | Rubrics, benchmark alignment, and regression tasks. |

Generated repos should match or exceed the V1 scaffold benchmark before relying on the agent team.
