# /kb-promote

Use `obsidian-project-kb-core` to decide whether content should become:
- `Knowledge/`
- `Experiments/`
- `Results/`
- `Results/Reports/`
- `Writing/`

Do not promote raw session noise. Update `_system/registry.md` and `02-Index.md` after promotion.
