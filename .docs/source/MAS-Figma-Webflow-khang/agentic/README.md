# Agentic System

This folder stores durable agentic design, policies, memory, orchestration contracts, evals, and generated specs.

| Folder | Purpose |
|---|---|
| `specs/` | Seed input, `agent_system_spec`, scaffold file plan, artifact contracts, and validation report. |
| `schemas/` | JSON schema contracts for generated workspace artifacts. |
| `knowledge/` | Stable project knowledge and source maps. |
| `memory/` | Memory candidates and approved memory cards. |
| `policies/` | Tool risk, approval, no-overwrite, and MCP auth rules. |
| `orchestration/` | Workflow, handoff, retry, reflection, phase, and stop-condition contracts. |
| `evals/` | Standalone baseline, quality rubric, reflection rubric, golden tasks, and regression checks. |

This folder is the V3 agentic quality layer on top of the existing MAS workflow.
