# Project Overview

MAS-Figma-Webflow-khang is a multi-agent workflow for turning Figma designs into Webflow pages.

Core project traits:

- PM-led orchestration.
- Architect-owned Client-First blueprint and QA.
- Operator-owned Figma extraction and Webflow native build execution.
- Workspace-driven state through JSON files.
- Approval gate between blueprint and Webflow build.
- MCP-352 micro-chunking for Webflow operations.
- Pixel-perfect QA against actual Webflow state.
