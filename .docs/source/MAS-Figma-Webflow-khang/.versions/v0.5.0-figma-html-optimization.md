## Key CF Doc Findings Applied

- Stacking limit: 4 = absolute max (not "3-4"); 5+ always = merge to custom class
- Borders: 1px stays as 1px literal (not converted to rem)
- CF has NO flex/grid layout utilities — every layout is a new custom class
- Spacing wrapper pattern: wrap child in div with margin-* utilities, not margin on content element
- Section visual variants: use add-on combo class (`section-style-dark`), not inline styles on section_[name]
- Page-specific vs global prefix: page slug only when element never shared across pages
- Figma variable lookup is mandatory before HTML contract — STOP if token not in library
