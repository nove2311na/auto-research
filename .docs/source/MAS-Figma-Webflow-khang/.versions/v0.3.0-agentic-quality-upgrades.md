# v0.3.0 Agentic Quality Upgrades

Date: 2026-06-02

## Improvements

- Added a bounded reflection loop in `agentic/orchestration/reflection-loop.md`.
- Added a reflection scoring rubric in `agentic/evals/reflection-rubric.md`.
- Added the `reflection-review` Claude skill with references, script support, and an asset template.
- Added ReAct requirements for Webflow action logs: reason, action, observation, and next decision.
- Added JSON schemas for metadata, blueprint, state, error logs, page structure, QA reports, and phase state.
- Added a structured Client-First class library for mapping Figma properties to Webflow classes.
- Added a visual QA evidence contract that requires blueprint path, Webflow state reference, mismatch list, verdict, and fix owner.
- Added a phase state machine that blocks Webflow build without explicit user approval.
- Added gates for skill anatomy, workspace artifacts, phase state, and relative paths.
- Added `.versions/` so feature and improvement history stays visible but does not clutter the root.
