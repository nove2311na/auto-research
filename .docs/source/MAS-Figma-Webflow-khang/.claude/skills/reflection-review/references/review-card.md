# Reflection Review Card

Use this compact card for each artifact under review.

| Field | Value |
|---|---|
| Artifact | Relative path |
| Phase | Current phase |
| Evidence read | Relative paths or MCP observations |
| Score | 1.0 to 5.0 |
| Verdict | pass, revise, or block |
| Required fix | Specific fix or empty for pass |
| Owner | PM, architect, operator, steward, or gatekeeper |
| Stop condition | Threshold, repeated failure, or missing approval |
