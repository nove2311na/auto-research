# Documentation Index

This folder contains project documentation and reference material.

## Main Areas

- `runbooks/` — user-facing operating guides.
- `agentic/` — source specs for agents, skills, validation, and templates. Kept at this path because `.claude/*` files reference it.
- `version/` — versioned implementation notes and change reports.
- `reports/` — audits and review reports.
- `plans/` — design proposals and future implementation plans.
- `source/` — upstream papers, PDFs, and external reference repositories.
- `reorg/` — cleanup plans, notes, and verification reports.
